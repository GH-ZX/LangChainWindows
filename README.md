# نظام التحليل الذكي - وزارة الدفاع

تطبيق ذكي للاستعلام من قواعد البيانات باستخدام الذكاء الاصطناعي.

## متطلبات النظام

- Python 3.8+
- PostgreSQL
- Ollama

## التثبيت والإعداد

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. إعداد Ollama
```bash
# تثبيت Ollama من https://ollama.ai

# تشغيل الخدمة
ollama serve

# تحميل النموذج
ollama pull llama3
```

### 3. إعداد قاعدة البيانات
- تأكد من تشغيل PostgreSQL
- أنشئ قاعدة بيانات باسم `company_db`
- الإعدادات الافتراضية:
  - Host: localhost
  - Port: 5432
  - Database: company_db
  - User: postgres
  - Password: admin

### 4. التحقق من النظام
```bash
python check_system.py
```

## تشغيل التطبيق

```bash
python app.py
```

## استكشاف الأخطاء

### إذا توقف التطبيق عند "جاري معالجة الاستعلام":

1. **تحقق من Ollama:**
   ```bash
   ollama list
   ollama serve  # في terminal منفصل
   ```

2. **تحقق من النموذج:**
   ```bash
   ollama pull llama3
   ```

3. **تحقق من قاعدة البيانات:**
   - تأكد من تشغيل PostgreSQL
   - تحقق من الإعدادات في التبويب "الإعدادات والاتصال"

4. **استخدم أداة التحقق:**
   ```bash
   python check_system.py
   ```

### ميزات التطبيق

- **واجهة عربية كاملة**
- **انميشن سلس للانتقال**
- **زر إلغاء لإيقاف العمليات**
- **رسائل خطأ واضحة مع حلول**
- **مهلة زمنية للعمليات (45 ثانية)**

## الدعم

إذا واجهت مشاكل، تحقق من:
- رسائل الخطأ في وحدة التحكم (Console)
- ملف `check_system.py` للتشخيص
- إعدادات قاعدة البيانات والـ Ollama
